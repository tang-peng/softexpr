实验1-1c. 引用(&)

C语言的指针强大而难用。C++引入了更安全的引用，其用法是：类型 &引用名=变量名。引用可视为某变量的一个别名；对引用操作与对变量本身操作效果一样。

实验说明：

  • 请新建命令行应用，执行如下程序

    xxxxxxxxxx

    1

    2
    int b = a;  // b是a的拷贝，即b是又一个int，其值与a相同
    3
    int& c = a; // c是a的引用，因此c就是a，仅是称呼不同
    4
    std::cout << "a:" << a << ",b:"<< b << ",c:" << c << std::endl;
    5
    b += 7;
    6
    std::cout << "a:" << a << ",b:"<< b << ",c:" << c << std::endl;
    7

    8

实验1-1e. C++动态内存

程序运行时，可能需动态地申请内存空间（所谓动态，指预先不知需多少内存空间）。C++提供了new和delete操作符代替C语言的malloc和free函数，前者用法更简单。其基本用法是：类型* 指针=new 类型和delete 指针。

注意：new和delete必须成对，以防内存泄漏！

实验说明：

  1. 执行以下程序。

    xxxxxxxxxx
    14
    1
    // 示例1：申请一段int大小的内存
    2
    int* x = new int; 
    3
    // 示例2：申请一个点，包含x，y坐标
    4
    struct Point { int x, y; };
    5
    Point* p = new Point; // 申请一段Point大小的内存
    6
    // 示例3：申请一段n个int大小的内存，其中n由键盘输入
    7
    int n;
    8
    std::cin >> n;
    9
    int* xs = new int[n]; 
    10
    // 注意：new和delete必须成对！
    11
    delete x;          // 释放申请的内存
    12
    delete p;          // 释放申请的内存
    13
    delete[] xs;       // 批量释放申请的内存
    14
    x = p = xs = NULL; // 将指针置为空

参考实验1-1d. C++函数，尽管允许函数重名，但函数参数从int变为float时，仍需重新定义函数。若考虑更多参数类型，重定义函数将很繁琐。这种情况下，适合用模板（template）代替不同参数类型。

使用模板的示例如下：

xxxxxxxxxx
6
1
// 定义
2

3
T square(const T& val) {return val*val;} // 在数据类型T的辅助下，给出统一的函数实现
4
// 使用
5
int a = square<int>(5);          // T 被替换为 int，a 的值是 25
6
double b = square<double>(5.0);  // T 被替换为 double，b 的值是 25.0
